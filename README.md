# Sales-Analysis-Dashboard
A *web-based interactive dashboard* designed to track, visualize, and analyze key sales metrics for tech sales teams. This project demonstrates how HTML, CSS, JavaScript, and Chart.js can be used to build an intuitive and dynamic sales tracking tool.

## Features
- Lead Tracking: Input leads from multiple channels including LinkedIn, Email, and Referrals.
- Closed Deals Tracking: Record the number of deals closed.
- Dynamic Metrics Calculation:
  - Total leads
  - Conversion rate
  - Estimated revenue
  - Interactive Charts:
  - Sales Chart: Bar chart showing leads per channel.
  - Revenue Chart: Line chart displaying revenue trends over time.
- Responsive Design: Works on desktops and mobile devices.
- User-Friendly Interface: Simple inputs, immediate updates, and visually appealing charts.

- ## Technologies Used
- HTML – structure of the dashboard.
- CSS – styling and layout.
- JavaScript – calculations, dynamic content, and chart updates.
- Chart.js – interactive and responsive charts.

- ## How It Works
1. Enter the number of leads from LinkedIn, Email, and Referrals in the input fields.
2. Enter the number of closed deals.
3. The dashboard will calculate:
   - Total leads
   - Conversion rate (closed deals ÷ total leads × 100)
   - Estimated revenue (closed deals × revenue per deal)
4. The Sales Chart updates dynamically to show the number of leads from each channel.
5. The Revenue Chart displays revenue trends for the months or periods you input.
